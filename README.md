Movie Watch List

A responsive web application for searching movies and creating a personal watchlist using the OMDb API.

Features
Search movies by title
View movie details
Add movies to a watchlist
Remove movies from the watchlist
Save data with localStorage
Responsive design
Technologies
HTML5
CSS3
JavaScript (ES6+)
Fetch API
LocalStorage
OMDb API
